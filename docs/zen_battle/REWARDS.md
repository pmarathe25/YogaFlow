| 🟢 Aversion Healed | Defeat Dvesha |

## Trophies

Earned by meeting special conditions.

| Trophy | Name | Condition |
|--------|------|-----------|
| 🥈 Fearless | Defeat Bhaya without any hero below 50% HP |
| 🥈 Anxiety Breaker | Defeat Chinta within 3 rounds |
| 🥈 Rage Tamer | Defeat Krodha before it berserks |
| 🥈 Unshaken | Defeat Ahankara without any hero dying |
| 🥈 Speed Demon | Defeat any monster on the first round |
| 🥇 True Calm | Collect all basic badges |
| 🥇 Combo Master | Use all 2-hero combos at least once |
| 🥇 Perfect Run | Beat Klesh without using any combo or ultimate |
| 💎 Perfect Harmony | Beat all 4 bosses with the same party, no deaths |
| 💎 Trinity | Use all 3-hero combos in a single battle |
| 💎 Zen Master | Collect all badges + all trophies |

## Monster Unlock Progression

Beat all Easy (4) → unlock Medium (3) → Beat Medium → unlock Hard (5) → Beat Hard → unlock Bosses → Beat Klesh → unlock Samsara.
